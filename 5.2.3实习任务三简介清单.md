- 解决碰撞模型的 Bug: 速度相同、方向接近的小球碰撞后，简单交换速度和角度，会出现“粘球”现象  

	```cpp
			if(fabs (angle-b.angle)<3){
				this->editXY (this->radius, this->radius);
				b.editXY (-b.getRadius (),-b.getRadius ());
			}
			if(fabs(b.speed-speed)<1){
				this->editXY (this->radius, this->radius);
				b.editXY (-b.getRadius (),-b.getRadius ());
			}
	```

	当两个球满足条件时, 直接修改两球的位置, 使其分开一段距离
